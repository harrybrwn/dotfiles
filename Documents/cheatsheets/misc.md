Commands
--------
- `locate` and `mlocate` are faster alternatives to the find command.
- `updatedb` updates what locate and mlocate are able to see

| name | action           |
| ---  | ----             |
| cd   | change directory |

grep for crlf ("\r\n") line endings: `grep -IUlr $'\r' **/*`


Kill a suspended process => run `$ fd`
